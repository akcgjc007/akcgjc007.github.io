It is an ethereum based crowdfunding application that solves the trust problem between a manager and contributors. Safe elections: A tamper free election service based on Ethereum

Safe elections app is a voting app based on ethereum smart contracts and made with Node.js, React.js, and Semantic UI React. The app provides security by only allowing restricted permissions to both voting and organizing parties. Because it is a decentralized application, it resides on the blockchain and cannot be hacked or changed.
